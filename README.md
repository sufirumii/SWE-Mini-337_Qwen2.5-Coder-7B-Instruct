# SWE-Mini-337_Qwen2.5-Coder-7B-Instruct

A small, synthetically generated dataset of software engineering tasks generated by Qwen2.5-Coder-7B-Instruct.

## Overview

This dataset contains 337 software engineering tasks, each paired with a step-by-step decomposition plan and a complete Python implementation. Every sample was generated by prompting **Qwen2.5-Coder-7B-Instruct** to invent a task in a given domain and write the full working code for it in a single pass.

It is intended for experimentation with supervised fine-tuning of small language models on coding tasks, not as a large-scale or clinically evaluated training set.

## Dataset Summary

- **Samples**: 337
- **Generator model**: Qwen2.5-Coder-7B-Instruct (4-bit NF4 quantized)
- **Format**: JSONL, one JSON object per line
- **License**: MIT

## Domains Covered

Samples are spread across 20 software engineering domains:

1. Networking and HTTP clients
2. Data structures and algorithms
3. File system and I/O operations
4. Concurrency and async programming
5. Database and ORM patterns
6. Caching systems
7. Authentication and security
8. Message queues and event systems
9. CLI tools and argument parsing
10. Serialization and data formats
11. Testing frameworks and mocking
12. Logging and monitoring
13. Web scraping and parsing
14. API design and REST patterns
15. Configuration management
16. Dependency injection
17. Stream processing
18. Graph algorithms
19. String processing and parsing
20. Numerical computing

## Fields

| Field | Description |
|---|---|
| `id` | Unique sample identifier |
| `task_title` | Short descriptive title of the task |
| `task_description` | One-paragraph description of the requirements |
| `domain` | Engineering domain the task belongs to |
| `complexity` | Self-reported difficulty: moderate, complex, or expert |
| `decomposition_steps` | Ordered steps breaking the task down |
| `implementation` | Complete Python implementation |
| `edge_cases` | Edge cases the implementation should handle |
| `test_cases` | Example test cases for the implementation |
| `source` | Generator model ID |
| `created_at` | Generation timestamp |

## Limitations

- Small scale — 337 samples is not enough to train a model on its own, and is best used as a seed or experimentation set.
- Generated by a single 7B model, so style and failure patterns are not diverse.
- Implementations were checked for valid Python syntax, but not executed against their own test cases, so some logic errors may remain.
